## 配置LLM API

要使用真实的AI回复（而非模拟），需要配置LLM API Key。

### 支持的API

| 提供商 | 默认模型 | 费用 |
|--------|----------|------|
| OpenAI | gpt-4o-mini | 按量付费 |
| 任何兼容OpenAI接口的服务 | 可配置 | 按服务商而定 |

### 配置方式

#### 方式一：环境变量（推荐）

**Windows (PowerShell):**
```powershell
$env:OPENAI_API_KEY = "sk-your-api-key-here"
```

**macOS / Linux:**
```bash
export OPENAI_API_KEY="sk-your-api-key-here"
```

#### 方式二：修改配置文件

编辑 `config.py`，找到 `LLMConfig` 部分：

```python
@dataclass
class LLMConfig:
    api_key: str = "sk-your-api-key-here"  # 填入你的API Key
    api_base: str = "https://api.openai.com/v1"
    model: str = "gpt-4o-mini"
```

### 使用国内模型服务

如果你使用国内的模型服务（如 DeepSeek、智谱、通义千问等），只需修改 `api_base`：

```python
# DeepSeek 示例
api_key: str = "sk-your-deepseek-key"
api_base: str = "https://api.deepseek.com/v1"
model: str = "deepseek-chat"
```

## API接口说明

### 1. 发送消息（被动链路）

**端点：** `POST /api/agent/message`

```bash
curl -X POST http://localhost:8000/api/agent/message \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "user_001",
    "message": "我下周有一个项目汇报，需要准备PPT",
    "session_id": ""
  }'
```

**响应：**
```json
{
  "reply": "收到！我会帮您记录会议信息...",
  "chain": "passive",
  "memory_writes": [
    {"type": "event", "content": "用户提到：项目汇报...", "id": "mem_xxx"}
  ],
  "session_id": "ses_xxx",
  "timestamp": "2026-07-03T10:00:00"
}
```

### 2. 检查主动推送（主动链路）

**端点：** `GET /api/agent/push/check?user_id=user_001`

```bash
curl "http://localhost:8000/api/agent/push/check?user_id=user_001"
```

**响应（有推送时）：**
```json
{
  "has_push": true,
  "push": {
    "message": "您今天下午有会议，需要准备资料吗？",
    "push_type": "event_reminder",
    "priority_score": 0.85,
    "cost_score": 0.3,
    "chain": "active"
  }
}
```

### 3. 推送反馈

**端点：** `POST /api/agent/push/feedback`

```bash
curl -X POST http://localhost:8000/api/agent/push/feedback \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "user_001",
    "push_timestamp": "2026-07-03T10:00:00",
    "accepted": true
  }'
```

### 4. 检查Drift探索

**端点：** `GET /api/agent/drift/check?user_id=user_001`

```bash
curl "http://localhost:8000/api/agent/drift/check?user_id=user_001"
```

### 5. 查询记忆

**端点：** `GET /api/agent/memory/{user_id}`

```bash
# 查询所有记忆
curl "http://localhost:8000/api/agent/memory/user_001"

# 按关键词搜索记忆
curl "http://localhost:8000/api/agent/memory/user_001?query=项目汇报"

# 限制返回数量
curl "http://localhost:8000/api/agent/memory/user_001?limit=5"
```

### 6. 删除记忆

**端点：** `DELETE /api/agent/memory/{user_id}/{memory_id}`

```bash
curl -X DELETE "http://localhost:8000/api/agent/memory/user_001/mem_xxx"
```

### 7. 隐私同意

```bash
# 记录同意
curl -X POST http://localhost:8000/api/agent/consent \
  -H "Content-Type: application/json" \
  -d '{"user_id": "user_001", "purposes": ["对话记忆", "个性化推荐"]}'

# 撤回同意（会删除所有记忆）
curl -X DELETE "http://localhost:8000/api/agent/consent/user_001"
```

### 8. 调用插件

```bash
# 天气查询
curl -X POST http://localhost:8000/api/agent/plugin \
  -H "Content-Type: application/json" \
  -d '{
    "plugin_name": "weather",
    "action": "get_activity_suggestion",
    "params": {"city": "北京"}
  }'

# 日历添加事件
curl -X POST http://localhost:8000/api/agent/plugin \
  -H "Content-Type: application/json" \
  -d '{
    "plugin_name": "calendar",
    "action": "add_event",
    "params": {
      "user_id": "user_001",
      "title": "项目汇报",
      "datetime": "2026-07-10T14:00:00",
      "reminder_minutes": 30
    }
  }'
```

### 9. 查看审计日志

```bash
curl "http://localhost:8000/api/agent/audit/user_001"
```

### 10. 系统状态

```bash
curl "http://localhost:8000/status"
```

## 配置说明

所有配置项在 `config.py` 中，主要分为以下几个部分：

### LLM模型配置

```python
@dataclass
class LLMConfig:
    provider: str = "openai"           # 模型提供商
    api_key: str = ""                  # API密钥
    api_base: str = "https://api.openai.com/v1"  # API地址
    model: str = "gpt-4o-mini"         # 模型名称
    max_tokens: int = 2048             # 最大生成长度
    temperature: float = 0.7           # 创造性（0-1）
```

### 记忆系统配置

```python
@dataclass
class MemoryConfig:
    max_context_memories: int = 10     # 每次检索最多返回的记忆数
    compression_threshold: int = 50    # 对话超过50轮触发压缩
    compression_keep_recent: int = 20  # 压缩时保留最近20轮
    versioning_enabled: bool = True    # 记忆版本控制
    encryption_enabled: bool = False   # 是否加密（生产环境应开启）
```

### 主动推送配置

```python
@dataclass
class PushConfig:
    max_daily_pushes: int = 10         # 每日最大推送次数
    min_interval_seconds: int = 300    # 两次推送最小间隔（5分钟）
    quiet_hours_start: int = 22        # 免打扰开始（晚上10点）
    quiet_hours_end: int = 8           # 免打扰结束（早上8点）
    default_cost_threshold: float = 0.5 # 成本阈值
```

### Drift探索配置

```python
@dataclass
class DriftConfig:
    idle_threshold_seconds: int = 3600 # 静默1小时触发
    cooldown_seconds: int = 7200       # 两次Drift最小间隔2小时
    max_daily_drifts: int = 5          # 每日最大探索次数
    confidence_threshold: float = 0.6  # 新信息写入置信度阈值
```

### Agent人格配置

```python
@dataclass
class AgentConfig:
    name: str = "小艾"                 # Agent名称
    persona: str = "你是一个温暖、善解人意的AI好友..."
    tone: str = "友好、自然、共情"
    traits: list = ["善于倾听", "主动关怀", "幽默风趣", "知识渊博"]
```

## 常见问题

### Q1: 安装依赖时报错

**问题：** `pip install` 失败或超时

**解决：** 使用国内镜像源

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q2: 端口被占用

**问题：** `Address already in use`

**解决：** 修改 `config.py` 中的端口号：

```python
class AppConfig:
    api_port: int = 8001  # 改为其他端口
```

### Q3: 如何重置所有记忆？

删除 `data/memories/` 目录下的JSON文件，或调用API：

```bash
curl -X DELETE "http://localhost:8000/api/agent/consent/user_001"
```

### Q4: 没有LLM API Key怎么办？

使用演示模式，直接运行 `python demo.py`，无需任何API Key。

演示模式内置了模拟回复，覆盖了常见场景（会议、面试、提醒、偏好、天气、健康、学习、压力等）。

### Q5: 如何用自己的LLM模型？

修改 `config.py` 中的LLM配置，支持任何兼容OpenAI接口的服务：

```python
# 例如使用本地部署的Ollama
api_key: str = "ollama"
api_base: str = "http://localhost:11434/v1"
model: str = "llama3"
```

### Q6: 数据存储在哪里？

- 记忆文本：`data/memories/{user_id}.json`
- 向量数据：`data/vectors/vectors.json`
- 审计日志：`data/logs/audit.log`
- 应用日志：`data/logs/app.log`
- 同意记录：`data/memories/consent_records.json`

### Q7: 如何添加自定义插件？

在 `plugins/` 目录下创建新文件，继承 `BasePlugin`：

```python
from plugins.base_plugin import BasePlugin, PluginPermission

class MyPlugin(BasePlugin):
    name = "my_plugin"
    description = "我的自定义插件"
    permissions = PluginPermission(
        read_memory=True,
        send_notification=True
    )
    
    def execute(self, action: str, params: dict) -> dict:
        if action == "my_action":
            return {"success": True, "result": "Hello!"}
        return {"success": False, "error": "未知操作"}
    
    def get_actions(self) -> list[str]:
        return ["my_action"]
```

然后在 `main.py` 的 `_register_default_plugins` 方法中注册：

```python
self.plugin_registry.register(MyPlugin())
```

## 快速测试脚本

将以下内容保存为 `test_api.py`，启动API服务后运行：

```python
import requests

BASE = "http://localhost:8000"
USER = "test_user"

# 1. 同意隐私
r = requests.post(f"{BASE}/api/agent/consent", json={
    "user_id": USER,
    "purposes": ["对话记忆"]
})
print("1. 隐私同意:", r.json())

# 2. 发送消息
r = requests.post(f"{BASE}/api/agent/message", json={
    "user_id": USER,
    "message": "我下周有个面试，需要准备。"
})
print("2. 发送消息:", r.json()["reply"])

# 3. 再次发送消息
r = requests.post(f"{BASE}/api/agent/message", json={
    "user_id": USER,
    "message": "另外我喜欢跑步和游泳。"
})
print("3. 发送消息:", r.json()["reply"])

# 4. 查询记忆
r = requests.get(f"{BASE}/api/agent/memory/{USER}")
memories = r.json().get("memories", [])
print(f"4. 记忆数量: {len(memories)}")
for m in memories:
    print(f"   [{m['type']}] {m['content'][:60]}")

# 5. 检查推送
r = requests.get(f"{BASE}/api/agent/push/check", params={"user_id": USER})
print("5. 推送检查:", r.json())

# 6. 调用天气插件
r = requests.post(f"{BASE}/api/agent/plugin", json={
    "plugin_name": "weather",
    "action": "get_activity_suggestion",
    "params": {"city": "北京"}
})
if r.json().get("success"):
    print("6. 天气:", r.json()["result"]["tip"])

# 7. 系统状态
r = requests.get(f"{BASE}/status")
print("7. 状态:", r.json())
```
